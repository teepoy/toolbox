Dataset Manager — manage your ML datasets

Usage: dman-cli <COMMAND>

Commands:
  init         Initialize the dman catalog (~/.dman or $DMAN_HOME)
  add          Register a dataset with the catalog
  list         List all registered datasets
  inspect      Show detailed information about a dataset
  remove       Remove a dataset from the catalog (does not delete files)
  import       Import a dataset from a path (not yet implemented)
  export       Export a dataset to a path (not yet implemented)
  operate      Apply operations to a dataset (not yet implemented)
  virtual      Manage virtual datasets (not yet implemented)
  serve        Start the dman HTTP server (not yet implemented)
  tui          Launch the terminal UI (not yet implemented)
  materialize  Materialize a virtual dataset (not yet implemented)
  help         Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
