## API Routes Added
GET /api/datasets                     → list_datasets (paginated)
GET /api/datasets/{name}              → get_dataset (404 if not found)
GET /api/datasets/{name}/images       → list_images (?category=X&page=N&per_page=N)
GET /api/datasets/{name}/images/{id}  → get_image + annotations
GET /api/datasets/{name}/categories   → list_categories
GET /api/datasets/{name}/stats        → get_dataset_stats
GET /api/virtual-datasets             → list_virtual_datasets (paginated)
GET /api/virtual-datasets/{name}      → get_virtual_dataset (404 if not found)

## Response Format
Success: { "data": ..., "pagination": { "page", "per_page", "total" } }
Error:   { "error": { "code": "NOT_FOUND", "message": "..." } }
