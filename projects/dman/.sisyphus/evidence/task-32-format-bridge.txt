## Key Design Decisions
- Structs always compiled; `python_impl` module with trait impls behind `#[cfg(feature = "python")]`
- `info` field has `#[allow(dead_code)]` when python feature is off
- `with_plugin_module()` helper builds CStrings OUTSIDE Python::attach closure (borrow-checker requirement)
- `detect()` returns false if Python plugin has no `detect` function (graceful fallback)
- `import()` parses `{"images": [...], "annotations": [...]}` dict, wraps DB writes in BEGIN IMMEDIATE transaction
- `export()` queries images+annotations, builds Python dicts, calls `export_dataset(data, output_path)`
