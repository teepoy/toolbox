# dman Docs

This directory introduces the main product features without forcing readers through a single long walkthrough.

## Start points

- [Quickstart](../quickstart.md) — end-to-end hands-on setup and examples
- [Catalog and CLI](./features/catalog-and-cli.md) — catalog model and everyday commands
- [Format registry](./features/format-registry.md) — built-in and user-defined dataset formats
- [Python SDK](./features/python-sdk.md) — programmatic dataset creation, loading, and updates
- [Label Studio](./features/label-studio.md) — annotation-tool integration
- [TUI](./features/tui.md) — interactive terminal workflow

## Recommended reading order

1. Quickstart
2. Catalog and CLI
3. Format registry
4. Python SDK or Label Studio, depending on workflow
5. TUI for interactive browsing
